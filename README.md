# Tech Store

## Description

The main purpose of this project is to practice creating web pages and build something completely functional. I'm currently using HTML and CSS, and later I'll add JavaScript for interactivity, React for the frontend, and a backend to practice working with APIs, databases, and full-stack development.

## Technologies

* HTML5
* CSS3

## Current Features

* Header with navigation bar
* Product cards
* Clean and simple layout
* Responsive design (in progress)

## Future Features

* Add more products
* Shopping cart functionality
* JavaScript interactivity (buttons, filters, etc.)
* React frontend
* Node.js + Express backend
* PostgreSQL database
* API connection between frontend and backend
* User authentication

## Future Plans

This project will keep growing as I learn new technologies. The idea is to turn it into a full-stack web application step by step, improving both the design and functionality over time.
